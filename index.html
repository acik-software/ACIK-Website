Açık Software

Acik Software was founded to create open source software that is easy to use.

We are students that wordking on software projects that allow us to help other and learn in the process.

We aim to help people by producing quality software that is open source and also easy to use, meaning that one does not need to learn technical information to use it.

Being students, we have a unique persepective as we are not a part of a big company.

Since the beginning we have always strived to others help by making our software open source and easy to use.
